quoteCtrl
controller in module emptyNgApp

Description

Creates the controller for the quote view

Dependencies

  • $scope
  • $location

Usage

quoteCtrl(quote, $scope, $location);

Parameters

ParamTypeDetails
quoteNumber

The quote which is returned form the quoteService

$scopeObject

The scope of the controller, used to access the view

$locationObject

Gives you access to the location object (used to manipulate the current path)
