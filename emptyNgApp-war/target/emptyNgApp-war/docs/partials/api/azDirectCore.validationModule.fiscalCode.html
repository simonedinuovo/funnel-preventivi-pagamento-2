fiscalCode
service in module azDirectCore.validationModule

Description

The fiscalCode contains function that help during develop with Date;

Methods

  • calculate(nome, surname, gg, mm, aa, place)

    This method give the Fiscal code, for example : 

    
    console.log(fiscalCode.calculate("mario", "rossi", 31, 05, 1992, "M", "H501");

    //will print "RSSMRA92E31H501J"
    Parameters
    ParamTypeDetails
    nomestring

    Person's name from which generate fiscal code

    surnamestring

    Person's surname from which generate fiscal code

    ggstring

    Person's day of birthdate from which generate fiscal code

    mmstring

    Person's month of birthdate from which generate fiscal code (NB: starting from 0 as month of Data js obj)

    aastring

    Person's year of birthdate from which generate fiscal code

    placestring

    Person's birthplace from which generate fiscal code

    Returns
    string

    Fiscal code

  • encode_comune(comune_pattern)
